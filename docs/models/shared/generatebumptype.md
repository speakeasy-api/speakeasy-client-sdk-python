# GenerateBumpType

Bump type of the lock file (calculated semver delta, custom change (manual release), or prerelease/graduate)


## Values

| Name         | Value        |
| ------------ | ------------ |
| `MAJOR`      | major        |
| `MINOR`      | minor        |
| `PATCH`      | patch        |
| `CUSTOM`     | custom       |
| `GRADUATE`   | graduate     |
| `PRERELEASE` | prerelease   |
| `NONE`       | none         |
